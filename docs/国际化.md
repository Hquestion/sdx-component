### 国际化

默认中文，会加载中文的资源文件。切换语言的API暂未实现

#### 配合vue-i18n

配合vue-i8n,可实现动态切换语言.具体方案如下：

- 导入资源文件，以ui为例

```
import enMsg from  '@iluvatar/ui/lib/utils/locale/lang/en';
```

- 在i18n注入Vue实例之后，调用i18n的API合并资源文件

```
const app = new Vue({
    router,
    store,
    i18n,
    render: h => h(App)
}).$mount('#app');

app.$i18n.mergeLocaleMessage('en', enMsg);
```
